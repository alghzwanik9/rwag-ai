## 🌟 نبذة عن المشروع (About)
**رواق (Rwaq-AI)** هو نظام متكامل يتيح للمصممين والمستخدمين بناء وتأثيث غرفهم ومساحاتهم بشكل افتراضي (ثلاثي الأبعاد) من خلال الأوامر النصية وتوليد الذكاء الاصطناعي.
يدمج المشروع بين تقديم تجربة بصرية مذهلة في المتصفح باستخدام `Three.js`، وبين محرك بحث ذكي للأثاث يعتمد على `LanceDB` و `FastAPI`.

---

## ✨ الخدمات والمميزات (Features)
- 🎨 **توليد التصاميم بالذكاء الاصطناعي:** تحويل الأوصاف النصية (Prompts) إلى توزيع أثاث ذكي داخل بيئة 3D.
- 🛋️ **كتالوج أثاث ديناميكي:** سحب وفهرسة قطع الأثاث ثلاثية الأبعاد (GLB) محلياً أو عبر كاشطات (Scrapers) متطورة وحفظها في قاعدة بيانات متجهات (Vector DB).
- 🌞 **محاكاة الإضاءة والطقس:** تغيير أوقات اليوم (صباح، ممطر، منتصف الليل) مع تأثيرات إضاءة واقعية.
- 🛠️ **أدوات تخصيص شاملة:** تغيير ألوان الجدران، الأرضيات، النوافذ، والستائر فورياً.
- 🔐 **مصادقة آمنة وحفظ المشاريع:** ربط النظام بـ `Clerk` للمصادقة وتجهيز `Supabase` لحفظ مشاريع المستخدمين في السحابة.

## 🛠️ التقنيات المستخدمة (Tech Stack)

### واجهة المستخدم (Frontend)
- **Next.js 16 (App Router)** - إطار العمل الأساسي.
- **React 19 & Tailwind CSS v4** - لبناء واجهات أنيقة وعصرية.
- **Three.js & React Three Fiber (R3F)** - لعرض البيئة ثلاثية الأبعاد.
- **Zustand** - لإدارة حالة المشهد والتصميم محلياً.

### الخادم (Backend)
- **FastAPI (Python)** - خادم سريع للذكاء الاصطناعي وإدارة البيانات.
- **LanceDB** - قاعدة بيانات متجهات (Vector Database) للبحث الذكي عن الأثاث.
- **Gemini API** - معالجة اللغة الطبيعية واستنتاج توزيع الأثاث.

## 🚀 طريقة التشغيل (How to Run)

للحصول على بيئة عمل محلية كاملة، يجب تشغيل الخادمين معاً.

### 1. تشغيل خادم الذكاء الاصطناعي (Backend)
في مجلد المشروع الرئيسي:
```bash
# إنشاء بيئة بايثون افتراضية (اختياري لكن مستحسن)
python -m venv venv
source venv/bin/activate  # في الويندوز: venv\Scripts\activate

# تثبيت الحزم المطلوبة
pip install "fastapi[standard]" uvicorn lancedb pyarrow httpx beautifulsoup4 python-dotenv

# تشغيل الخادم على المنفذ 8000
uvicorn main:app --reload
```

### 2. تشغيل واجهة المستخدم (Frontend)
افتح نافذة طرفية (Terminal) جديدة:
```bash
cd frontend

# تثبيت الحزم
npm install

# تشغيل خادم التطوير على المنفذ 3000
npm run dev
```

الآن يمكنك زيارة [http://localhost:3000](http://localhost:3000) للبدء في استخدام المنصة!

## 📂 هيكلية المشروع الأساسية
- `/frontend`: تطبيق Next.js بالكامل.
- `main.py`: خادم FastAPI الذي يربط الذكاء الاصطناعي بالتطبيق.
- `asset_scraper.py`: سكربت سحب وفهرسة الأصول ثلاثية الأبعاد.
- `.lancedb/`: مجلد قاعدة البيانات المحلية الخاصة بـ LanceDB.
