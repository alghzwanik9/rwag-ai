# 🚀 Atelier AI - Project Workflow

This document summarizes the progress, architecture, and completed tasks for the **Atelier AI** (Interior Design Assistant) project.

## 🏗️ Architecture Overview
The system relies on an **Orchestrator Pattern** to connect a modern Next.js frontend with a powerful 3D generation backend:
1. **Frontend**: Next.js 15, Tailwind CSS v4, React Three Fiber (R3F).
2. **Backend**: FastAPI (Python).
3. **3D Engine**: Blender Python API (Headless mode).
